# Credit-default-prediction-LogisticRegression-XGBoost
Credit default prediction pipeline — end-to-end ML on 150K borrowers. Covers data cleaning, outlier capping, median imputation, StandardScaler, VIF-based multicollinearity removal, SMOTE resampling, and F1-optimal threshold tuning. Compares Logistic Regression vs XGBoost with SHAP interpretability and post-hoc WoE/IV analysis.
